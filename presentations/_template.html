Bourne Again Shell (Bash)

Bash is a Unix command-line interpreter written by Brian Fox for the GNU Project as a free software replacement for the Bourne shell. It was first released in 1989, and now is the default login shell for most Linux distributions.

Other popular shells are Korn shell (ksh), Z shell (zsh), and the Friendly Interactive Shell (fish). Most of the topics we discuss in this presentation, also apply to these shells.
